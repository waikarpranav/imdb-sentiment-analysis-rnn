# IMDB Movie Review Sentiment Analysis (RNN)

This project performs sentiment analysis on IMDB movie reviews using a
Simple RNN model and deploys it using Streamlit.

## Tech Stack
- Python
- TensorFlow / Keras
- NLP
- Simple RNN
- Streamlit

## Features
- Classifies movie reviews as Positive or Negative
- Uses word embeddings for feature representation
- Interactive web interface using Streamlit

## How to Run

```bash
pip install -r requirements.txt
python -m streamlit run app.py
